# Setup sever

### Updating and restarting

1. ssh into your server
2. update software

```bash
apt update
apt upgrade
```

3. Restart your server

```
shutdown now -r
```

### Create new user

- root: highest permission level. Allows unrestricted access to OS.
- sudo: super user do. Allows you to run commands and programs as root.

#### Steps:

1. Create a new user `adduser <your_username>`

2. Add user to to "sudo" group `usermod -aG sudo <your_username>`

3. Switch user `su <your_username>`

4. Check sudo access `sudo cat /var/log/auth.log`

### Enable login as new user

1. Create authorized_keys file `~/.ssh/authorized_keys`
2. Paste your public key
3. Exit
4. Login with new user `ssh <your_username>@<your_IP>`

### Security

1. Change file permission `chmod 644 ~/.ssh/authorized_keys`
2. Disable root login `sudo vi /etc/ssh/sshd_config`
3. Restart ssh daemon `sudo service ssh restart or sudo systemctl restart ssh`

### Nginx setup

1. It is web server, can do reverse proxy, forward proxy.
2. Install nginx `sudo apt install nginx`
3. start nginx `sudo service nginx start` or `sudo systemctl reload nginx`
4. In ec2 you will need to add http in security group of ec2 instance.
5. View nginx config at `less /etc/nginx/sites-available/default`

### Application setup

1. Change ownership of /www `sudo chown -R $USER:$USER /var/www`
2. Make an application directory `mkdir /var/www/app`
3. Add new file in `/etc/nginx/sites-available`

```bash
server {
        listen 80 default_server;
        listen [::]:80 default_server;

        root /var/www/html;
        index index.html;

        server_name api.ankitesharora.com;

        location / {
                proxy_pass http://127.0.0.0:3000;
        }
}
```

### PM2

1. PM2 will keep running our node application even when we exist shell
2. Install PM2 `sudo npm i -g pm2`
3. Start PM2 `pm2 start app.js --watch`
4. Setup auto restart (if app goes down)

```
pm2 save
pm2 startup
```
