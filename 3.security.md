# Security

- Check auth logs, for checking how many bad actors are trying to get access to your server

```
sudo cat /etc/log/auth.log
```

- Port : Communication endpoint that maps specific process or network service
- See well known ports `less /etc/services`

### nmap to see open ports

1. Install nmap `sudo apt install nmap`
2. Run nmap `nmap <your_server_ip>`
3. Extra service/version information `nmap -sV <your_server_ip>`

### ufw (uncomplicated firewall)

1. ufw allow/deny/reject http/https/ssh
2. deny silently drops incoming network packets without replying,
   whereas reject explicitly send an error message

#### Setup

1. Check firewall status `sudo ufw status`
2. allow ssh `sudo ufw allow ssh`
3. allow http `sudo ufw allow http`
4. enable firewal `sudo ufw enable`

### Unattended updates

1. Install unattended upgrades `sudo apt install unattended-upgrades`
2. enable upgrades `sudo dpkg-reconfigure --priority=low unattended-upgrades`
